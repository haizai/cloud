css水平垂直居中 作者:haizai

  • 内联元素，已知容器高度，line-height居中

    inner1

    .inner1 {
  line-height : 100px; //已知容器高度为100px
  text-align  : center;
}

  • 内联元素，table居中

    inner2

    .outer2 {
  display : table;
}

    .inner2 {
  display        : table-cell;
  vertical-align : middle;
  text-align     : center;
}

  • 已知元素长宽，absolute + margin 居中

    inner3

    .outer3 {
  position : relative;
}

    .inner3 {
  position    : absolute;
  top         : 50%;
  left        : 50%;
  margin-left : -24px; // 已知元素宽度48px
  margin-top  : -0.5em;
}

  • 任意元素，absolute + translate居中(css3)

    inner4

    .outer4 {
  position : relative;
}

    .inner4 {
  position  : absolute;
  top       : 50%;
  left      : 50%;
  transform : translate(-50%,-50%);
}

  • 任意元素，flex居中(css3)

    inner5

    .outer5 {
  display         : flex;
  justify-content : center;
  align-items     : center;
}

  • 已知容器与元素长度，calc居中(css3)

    inner6

    .inner6 {
  margin-left : calc(50% - 24px);// 已知元素宽度48px;
  margin-top  : calc(50px - .5em);// 已知容器高度100px
}
